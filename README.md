Net-Model
---


本项目是深度学习论文复现，旨在收集整理经典深度学习模型，帮助学习者提高构建神经网络架构的能力。 \
本项目代码使用 Python 语言，基于 Pytorch 和 Kears 框架。另外还需要学习者了解基础的 NumPy 和 Matplotlib 操作。 \
本项目使用的数据集为论文中常用的经典数据集，例如 MNIST、Imagenet、COCO 等。

### 本项目内容包括：
- 论文原文
- 复现代码
- 扩展资料
