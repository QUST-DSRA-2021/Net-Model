本项目中的 AlexNet 使用 GPU 进行训练。 \
本项目上传了对应 ImageNet 数据集的网络结构和对应 MNIST 数据集的网络结构，并使用 MNIST 进行训练。 \
两种网络结构相同，根据不同数据集调整部分参数。
